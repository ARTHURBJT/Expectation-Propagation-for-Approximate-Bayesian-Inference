# Expectation-Propagation-for-Approximate-Bayesian-Inference

This project has been developed for the Bayesian machine learning course of the MVA master's degree.
It is based on the article : Thomas P. Minka. Expectation propagation for approximate bayesian inference, 2013. URL
https://arxiv.org/abs/1301.2294
You can run the experiments with the notebook, and read about what had been done on the pdf file.
